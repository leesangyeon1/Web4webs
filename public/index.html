Import bookmarks

A website can't read your browser's bookmarks directly — pick a file exported from the browser:

  1. Chrome export: open chrome://bookmarks → ⋮ (top-right) → Export bookmarks → save the .html.
  2. Or the raw file: choose Chrome's Bookmarks file (no extension) from your profile folder.

Sign in to sync

Pick an ID and passphrase. Use the same pair on another device to sync your bookmarks. New ID? It's created automatically.
